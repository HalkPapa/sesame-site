よく使う操作

実績を投稿する

実績 → 新規追加 → タイトル・本文・アイキャッチ画像を設定 → 公開

キャッシュをクリア

**Windows: Ctrl + Shift + R**

**Mac: Cmd + Shift + R**

色を変更する

外観 → テーマファイルエディター → style.css → カラーコードを検索・置換

カラーコード早見表

  -----------------------------------------------------------------------
  **色**                  **コード**              **用途**
  ----------------------- ----------------------- -----------------------
  黒                      #000000                 テキスト

  白                      #FFFFFF                 背景

  グレー                  #666666                 補足

  青（現在）              #5B9FBD                 メイン

  赤                      #C00000                 強調

  緑                      #00A651                 ボタン
